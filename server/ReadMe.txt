Если у Вас не установлен docker-compose, то установить его можно на сайте https://www.docker.com/ (можете сразу докер десктоп навалить, чтобы всё было и с интерфейсом)

Для работы потребуется либо дедик, либо правильно настроеный прокси на Вашем компе (всё в Ваших руках)

далее в консоль пишем docker-compose up, иногда если база подымается быстрее сервера, то сервер крашится, просто перезапустите его
этой же командой.

Для начала работы нужно прописать в консоль "npm install" без кавычек

Далее понадобится программа pgAdmin для работы c PostgreSQL
базой данных (линк: https://www.pgadmin.org/ в ней же качаем postgresql 15 (можно 14, если будет работать))
заходим в pgAdmin, ставим пароль от сервака 1234 (по дефолту) и от БД (1234 тоже, меняется в .env файле конфига в строке "DB_PASSWORD=1234")

Правой кнопкой мыши по слону и PostgreSQL *ваша версия* там выбираем create database, выбираем имя базы (в конфиге в строке "DB_NAME=data" его можно менять)
вписываем и ура победа

После этого пишем в консоль "npm run dev" (без кавычек) и получаем сообщение "server started on port 5000"
Порт, на котором открывается сервак также можно поменять в файле .env в строке "PORT=5000" если указанный Вами порт будет занят,
то по дефолту всё будет выполняться на 5000 порту
